This is a simple Cube Mapping demo made using OpengGL (fixed pipeline).
For license matters, consider the licence included on the repo's root folder.
Enjoy =)

CONTROLS
-MOUSE: rotation, zoom in/out
-KEYBOARD:
	- ' ' - change model
	- 'm' - change cube map
	- 's' - smooth normals
	- 'w' - wireframe/fill
	- Esc - exit
